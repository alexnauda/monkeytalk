-------------------------------------------------------------------------------
MonkeyTalk Java API
-------------------------------------------------------------------------------

This is the MonkeyTalk Java API.  You can build MonkeyTalk scripts and suites
directly in Java as JUnit test cases and test suites.  Then, you can run your
tests using any standard JUnit runner such as those provided by any decent
Java IDE (like Eclipse or IntelliJ) or any commandline Java build tool (like
Ant or Maven).

1. CONTENTS

This folder contains three jars:
1) monkeytalk-java-all-in-one.jar - the MonkeyTalk Java API (plus all its dependencies)
2) monkeytalk-java.jar - the MonkeyTalk Java API (without any dependencies)
3) monkeytalk-java-javadoc.jar - javadoc for the MonkeyTalk Java API
